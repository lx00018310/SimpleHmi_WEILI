## Primary Audience

- AI coding agents that need strict instructions before generating industrial HMI code.

## Important Boundaries

- Production chains must not rely on Mock, fake success, or silent fallback.
- Technical stack selection must be explained in the architecture stage.
- deployment_control is a capability to evaluate, not a universal default.
- Debug pages, debug APIs, and simulators must be isolated from formal production routes.
- Acceptance claims need evidence: build, run, config validation, interface checks, logs, rollback or non-rollback reasoning, and no-Mock verification.
