# 工业工位 HMI 示例页

这是 SimpleHmi_WEILI 的静态视觉示例，用于让用户直观看到按本规范生成的工业 HMI 页面风格。

## 打开方式

直接用浏览器打开：

```text
examples/industrial-workstation-demo/index.html
```

## 说明

- 本示例只展示界面布局、视觉密度、状态层级和操作组织方式。
- 本示例不提供后端接口，不代表正式数据链路。
- 正式项目仍必须按 `SIMPLE_HMI.md` 要求对接真实接口、真实状态和真实硬件信号。
- 如需调试模拟能力，必须按 `docs/09_去Mock规范.md` 独立隔离。
